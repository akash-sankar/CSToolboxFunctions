# ctrb

## Calling Sequence
- `C = pidstd (Kp)`
- `C = pidstd (Kp, Ti)`
- `C = pidstd (Kp, Ti, Td)`
- `C = pidstd (Kp, Ti, Td, N)`

## Description
Return the transfer function C of the PID controller in standard form with first-order roll-off.

## Parameters
- `Kp`: Proportional gain.
- `Ti`: Integral time.
- `Td`: Derivative time.
- `N`: First-order derivative filter divisor.
