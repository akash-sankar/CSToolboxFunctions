# @iddata/get

## Calling Sequence
- `get (dat)`
- `value = get (dat, 'key')`
- `[val1, val2, …] = get (dat, 'key1', 'key2', …)`

## Description
Return dimensions of iddata set dat.

## Parameters
- `val`: Key values of iddata objects.
- `dat` (Structure): iddata set.
- `key` (String): Key of the required value.
